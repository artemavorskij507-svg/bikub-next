# BiKuBe Next Goal

BiKuBe Next is a clean rebuild of BiKuBe as a production-grade local services operating system for Norway.

This project must become:
- Laravel 13 + Filament 5 admin/backoffice;
- PostgreSQL-first;
- modular-monolith architecture;
- real Order Engine;
- real Service Scenario Engine;
- real Dispatch Center;
- real Live Operations Map;
- real Worker/Courier PWA;
- real Customer Account;
- real Partner Portal;
- real Support system;
- real payments/payouts readiness;
- no fake GPS;
- no fake orders;
- no fake payments;
- no fake integrations.

The old project /home/keks/bikube must not be modified from this project.
Only proven domain logic should be ported later.
